## Frame 0：Drawer 容器（通用壳）

**Frame：`Drawer / TermDetail`（右侧抽屉）**

* **尺寸**：W 840（推荐 720–960 可配置） / H = Fill Screen
* **布局**：Auto Layout（Vertical）
* **层级**

  1. `Overlay / Scrim`（页面蒙层）
  2. `Drawer / Panel`

     * `Header / Sticky`
     * `Tabs / Sticky`
     * `Body / Scroll`
     * `Footer / Sticky (可选：仅编辑态显示)`

---

## Frame 1：Header（固定区）

**Frame：`Header / TermSummary`（Sticky）**

* **左侧**

  * `TitleGroup`

    * `H1 / TermNameCN`（如：自然人）
    * `Sub / TermNameEN`（如：Natural Person）
  * `MetaBadges`

    * `Badge / Status`（草稿/待评审/已发布/已废弃）
    * `Badge / Category`（业务对象/证照/技术术语…）
    * `Badge / Domain (可选)`（主题域/业务域）
* **右侧**

  * `PrimaryActions`

    * `Btn / Edit`（查看态显示）
    * `Btn / SubmitReview`（草稿/变更单态）
    * `Btn / Publish`（审批通过态）
    * `Btn / Deprecate`（已发布态）
  * `MoreActions / Dropdown`

    * 复制链接、复制TermId、导出、查看审计、（删除：仅草稿且无引用）
  * `Icon / Close`

> 交互：Header 固定；状态驱动按钮显隐与可用性。

---

## Frame 2：Tabs（固定区）

**Frame：`Tabs / TermDetailTabs`（Sticky）**

* `TabItem / Overview`（概览）
* `TabItem / Definition`（定义与口径）
* `TabItem / Relations`（关系）
* `TabItem / References`（引用）
* `TabItem / Workflow`（流程）
* `TabItem / Versions`（版本与变更）
* `TabItem / Comments`（评论）

> 建议：Tab 数可按 MVP 缩减（至少保留：概览/关系/引用/版本）。

---

## Frame 3：Body（滚动区）——两栏结构（推荐）

**Frame：`Body / Content`（Scroll）**

* **Auto Layout：Horizontal**

  * `MainColumn`（Fill Remaining）
  * `SideColumn / InfoAside`（W 280，Sticky within scroll 可选）

---

# A. 查看态（View Mode）

## A1：概览 Tab

**Frame：`Tab / Overview`（放在 MainColumn 内）**

1. `Section / KPI`

   * `Card / UsageCount`（使用次数）
   * `Card / ReferenceAssets`（引用资产数）
   * `Card / LastUpdated`（最近更新）
2. `Section / DefinitionPreview`

   * `Block / DefinitionText`（多行摘要 + “展开全文”跳转 Definition Tab）
   * `Block / ScopeTags (可选)`（适用范围/边界标签）
3. `Section / QuickRelations`

   * `List / SynonymsPreview`（同义词 chips，超过折叠）
   * `List / RelatedTermsPreview`（相关术语 chips，支持点击跳转）
4. `Section / QuickReferences`

   * `TablePreview / TopReferences`（Top 5 引用：资产类型/名称/绑定角色/更新时间）
   * `Link / ViewAllReferences`（跳转 References Tab）

**Frame：`Aside / InfoAside`（SideColumn 内）**

* `Card / BasicInfo`

  * TermId（可复制）
  * 分类、主题域（如有）
  * Owner / Steward
  * 创建人/创建时间、更新人/更新时间
* `Card / Governance`

  * 当前状态
  * 审批链/规则摘要（如：按分类路由）
  * 替代术语（仅废弃态显示）

---

## A2：定义与口径 Tab

**Frame：`Tab / Definition`**

* `Section / Definition`

  * `FieldView / DefinitionMarkdown`（支持 Markdown 渲染）
* `Section / Spec`

  * `FieldView / BusinessScope`（适用范围）
  * `FieldView / Boundary`（边界/不包含项）
  * `FieldView / Examples`（示例：字段/SQL/业务示例）
  * `FieldView / Notes`（备注/引用来源）
* `Section / Attachments (可选)`

  * `List / Links`（制度文档/标准链接）

---

## A3：关系 Tab（同义词/相关术语结构化）

**Frame：`Tab / Relations`**

1. `Section / Synonyms`

   * `Toolbar`

     * `Btn / AddSynonym`（打开选择术语弹窗/下拉）
   * `ChipsList / SynonymTerms`

     * Chip：术语名（点击跳转该术语抽屉）
     * Chip 操作：移除（受权限/状态控制）
   * `Hint / PreferredTerm`（可选：标记首选术语）
2. `Section / RelatedTerms`

   * `Toolbar`

     * `Btn / AddRelation`
     * `Filter / RelationType`（可选：上下位/组成/依赖/相关）
   * `RelationList`

     * Row：关系类型 + 术语 + 说明 + 操作（编辑/删除）
3. `Section / Hierarchy (可选，后续增强)`

   * `Tree / BroaderNarrower`（上位/下位）

---

## A4：引用 Tab（可钻取与影响评估）

**Frame：`Tab / References`**

* `FilterBar`

  * `Select / AssetType`（字段/表/业务对象/指标/规则/安全策略…）
  * `Select / BindingRole`（主绑定/候选/引用）
  * `Search / AssetKeyword`
  * `Select / EnvOrSystem (可选)`（数据源/系统）
* `Table / ReferenceList`

  * 列：资产类型｜资产名称｜路径｜绑定角色｜（置信度/来源 可选）｜更新时间｜操作
  * 操作：打开资产、解绑（权限+状态控制）
* `BatchActions (可选)`

  * 批量解绑、导出引用清单
* `ImpactPanel (可选)`

  * 删除/废弃提示：受影响资产数 + 下载清单

---

## A5：流程 Tab（生命周期/审批）

**Frame：`Tab / Workflow`**

* `Section / StatusTimeline`

  * 时间线：草稿→待评审→已批准→已发布→已废弃
* `Section / CurrentTask`

  * 当前节点、处理人、截止时间（如有）
  * 操作：通过/驳回/补充材料（权限控制）
* `Section / ApprovalHistory`

  * 表：节点｜处理人｜结果｜意见｜时间

---

## A6：版本与变更 Tab

**Frame：`Tab / Versions`**

* `SplitLayout`

  * `Left / VersionList`

    * 行：版本号｜状态｜更新时间｜变更人｜变更摘要
  * `Right / DiffViewer`

    * 对比：定义/关系/分类/Owner 等字段差异
    * 操作：恢复此版本（受权限+状态控制）
* `Btn / CreateChangeRequest`（已发布态修改入口，生成变更单）

---

## A7：评论 Tab

**Frame：`Tab / Comments`**

* `Composer / NewComment`

  * 输入框 + @人 + 附件（可选）
* `Thread / CommentList`

  * CommentCard：头像/姓名/时间/内容/回复/解决标记

# B. 编辑态（Edit Mode）

## B0：模式切换

**同一抽屉内部状态切换**（推荐）：

* Header 右侧：`Btn / SaveDraft`、`Btn / Cancel`
* Footer（可选）：放置主要保存按钮（移动端友好）

## B1：编辑表单（基础字段）

**Frame：`EditForm / BasicFields`（可作为 Definition Tab 的编辑态）**

* `Field / TermNameCN *`
* `Field / TermNameEN`
* `Select / Category`
* `Select / Domain (可选)`
* `Field / Definition *`（多行/Markdown Editor）
* `Field / Synonyms`（编辑态建议改为：可选术语 + 可输入新词）
* `Field / RelatedTerms`（关系选择器：术语 + 关系类型）

> 校验区（Inline Validation）：重名/同义词冲突提示（可链接到“疑似重复术语”列表弹窗）。

# C. 系统状态 Frame（必须预留）

1. **Loading**

   * `Skeleton / Header` + `Skeleton / Tabs` + `Skeleton / Body`
2. **Empty（无同义词/无引用/无版本）**

   * `EmptyState / Icon + Text + CTA`
3. **No Permission**

   * 字段只读 + 提示原因
4. **Blocked Action**

   * 例如：有引用禁止删除；已发布不允许直接编辑（引导创建变更单）
